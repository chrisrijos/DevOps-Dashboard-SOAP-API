# Installation
1. The first step is to make sure you have NPM installed on your computer by installing node.js at node.org
2. Make sure GIT is installed on your computer and run git clone https://github.com/chrisrijos/soapmessagetracker in the directory you wish to install this project.
3. CD into the soapmessagetracker directory you cloned from github.com in your shell and run npm install. --This should install necessary dependencies.
# Running the project
4. Once all the node package have been installed run "node server" in the shell from the root directory of the project.
5. The project should now be running at Localhost:8080
